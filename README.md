# superstore-sales-analysis
Exploratory data analysis of the Kaggle Superstore dataset using Python.

## Dataset
US retail store transaction data (2014–2017) with 9,994 rows, 21 columns.
Source: [Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

## What's in this repo
- `superstore_analysis.ipynb`: full analysis notebook
- `Sample - Superstore.csv: dataset

## Analysis covers
- Data cleaning and quality checks
- Univariate analysis of sales, profit, discount, and quantity
- Bivariate trends by region, category, and sub-category
- Multivariate month-by-month breakdown by category and year

## Key findings
- Technology is the most profitable category despite being the least ordered
- Tables, Bookcases, and Supplies are net loss-making sub-categories
- Revenue grew from $484K (2014) to $733K (2017) with consistent Q4 spikes across the years
- The West region leads on both volume and profit

## Tools used
Python: pandas, matplotlib, seaborn, plotly
